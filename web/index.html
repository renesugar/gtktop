Gtktop

What is Gtktop ?

Gtktop is a small OCaml library to ease the creation of graphical toplevels.

Gtktop was previously part of Cameleon (called "Custop") but is now developped separately and is findlib compatible.

Download

Gtktop is hosted on Github.

Releases

  • 2.0 [2012-04-12]: first release after extraction from Cameleon.

Installation

Gtktop requires

  • LablGtk >= 2.14.2, installed with ocamlfind,
  • LablGtk-extras >= 1.1.

To compile and install:

  • tar xvfz gtktop-XXX.tar.gz
  • cd gtktop-XXX
  • ./configure
  • make all install

This will install the findlib package "gtktop".
